## Описание программы
Book Tracker - это графическое приложение для управления списком прочитанных книг. Программа позволяет добавлять, удалять и фильтровать книги, а также сохранять данные в JSON формате.

### Основные возможности:
- Добавление книг с указанием названия, автора, жанра и количества страниц
- Валидация вводимых данных (проверка на пустые поля и корректность числа страниц)
- Фильтрация книг по жанру и количеству страниц
- Удаление книг из списка
- Сохранение данных в JSON файл
- Загрузка данных из JSON файла
- Автоматическое сохранение данных при закрытии программы

## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Операционная система: Windows, macOS или Linux

### Установка:
1. Скачайте файл `book_tracker.py`
2. Убедитесь, что Python установлен на вашем компьютере
3. Дополнительные библиотеки не требуются (используется только стандартная библиотека Python)

### Запуск:
```bash
python book_tracker.py
